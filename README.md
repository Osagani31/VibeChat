## 🌟 Overview

**Vibe Chat** is a full-stack real-time chat application built using the **MERN stack (MongoDB, Express, React, Node.js)**.

This app allows users to send and receive messages instantly without refreshing the page, using **Socket.IO** for real-time bi-directional communication.

## ✨ Features

- 🔐 User Authentication (Login / Signup)
- 💬 Real-time messaging with Socket.IO
- 🟢 Online / Offline user status
- 🔍 Search users
- 👤 Profile management
- ⚡ Instant message delivery (no reload)
- 📱 Responsive design (mobile + desktop)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Osagani31/vibe-chat.git
cd vibe-chat
```

### 2️⃣ Install dependencies
Frontend
cd client
npm install
Backend
cd server
npm install

### 3️⃣ Environment Variables

Create a .env file inside the server folder:
<br>
PORT=5000 <br>
MONGO_URI=your_mongodb_connection_string <br>
JWT_SECRET=your_secret_key <br>

### 4️⃣ Run the application
Start backend
cd server
npm run dev
Start frontend
cd client
npm start
<br>
